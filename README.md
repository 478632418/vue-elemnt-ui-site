# vue-element-ui
vue+element-ui实现的一个后台项目，包含登录，人员、角色、权限。基本上够自己以后弄一个后台时，参考来源使用。本项目根据学习网络教程过程中完成的一个学习项目。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
